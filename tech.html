Review of Recursion and its role in our computing world!

In this post, we're reviewing recursion and it's role and uses in our modern computing world.

What is recursion?

Recursion is a method where the solution for a problem can be derived from smaller processed instances of that same problem.

Some programs use functional programming languages that do not use looping to solve problems but instead use recursion to call and process code

These languages can solve the same problems as more commonly used imperative languages without using iteration constructs such as "while" or "for".

That being said, most languages do support both recursion and looping as acceptable code by allowing a function to call itself.

What does recursion do for computer programming?

A common recursion technique in programming is to divide a problem into sub-problems that are similar to the original.

Using this method, the divide-and conquer method, we solve the sub-problems and combine the results.

If we store the results of these subproblems in a table, this is then called dynamic programming or memoization, which, is an optimization technique used to pulled stored results from a cache of results from complex calculations.

Are there different types of recursions?

Recursions that contain only a single self reference is a single recursion. Similarly, recursions with multiple self references are known as multiple recursions.

Common single recursions types in the modern world are linear searches, factorial compuatations, and processing list traversals.

Common multiple recursions types are depth first searches and tree traversal (which is a process that checks and processes nodes in a tree data structure) a single time.).

Additionally, there are also direct and indirect recursions.

The most common type of recursion are direct recursions. Direct recursions are where the functions calls itself.

Indirect recursions occurs when a function is not called by itself but by another function. Indirect recursions are also known as mutual recursions.
