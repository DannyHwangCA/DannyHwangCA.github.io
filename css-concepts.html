This week I learned about how CSS changes the html code we created last week.

What is the difference between margin, border, and padding?

Margin is the outer most layer of a object on html. We mainly use margin to control the placement of the object on our page.

Border is the layer between the margin and the padding of an object (middle later). We mainly use border to create a border around the object.

Padding is the layer closest to the object's contents and acts as a space between the object and the border. We mainly use padding to create space for buttons or negative space for separation purposes.

What are the best practices associated with using classes vs. ids?

Each element can have only one ID and each page can only have one element with that associated ID.

But, an element on our page can have multiple classes and the class can be shared amongst many elements.

This means that classes are more flexible than IDs. But, we use IDs for more set css on our page. Like, the footer section's css or the header's css. Sections taht we can good set css in, We should us ID.

Classes can be used for many elements and it makes the class css our most used asset.

What are the differences between relative, absolute, fixed, and static positioning?

Relative positioning means the object will be positioned in relation to the object before it.

Absolute positioning means the object will be positioned in relation to the location where it should naturally be placed.

Fixed positioning means that the object will be fixed to a certain location and will not be affected by the browser's size/movements.

Static positioning is the default positioning of all objects and means the object will be placed in the original order it was coded.

What does it mean to display inline vs using inline block?

Display inline and display inline-block are exactly the same except that display:inline-block can accept a width and height modification.

This means that we can use display:inline-block as an actual block/boxed area, whereas; we cannot do that with display:inline.
