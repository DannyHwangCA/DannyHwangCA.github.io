Review of Recursion and its role in our computing world!

In this post, we're reviewing recursion and it's role and uses in our modern computing world.

What is recursion?

Recursion is a method where the solution for a problem can be derived from smaller processed instances of that same problem.

Some programs use functional programming languages that do not use looping to solve problems but instead use recursion to call and process code

These languages can solve the same problems as more commonly used imperative languages without using iteration constructs such as "while" or "for".

That being said, most languages do support both recursion and looping as acceptable code by allowing a function to call itself.

How does recursion relate to computer programming?

A common recursion technique in programming is to divide a problem into sub-problems that are similar to the original.

Using this method we solve the sub-problems and combine the results.
